## 📖 Sobre o Projeto

Este repositório contém o código-fonte da minha plataforma de autoridade e portfólio profissional. Mais do que um "site de desenvolvedor", esta é uma infraestrutura otimizada para captação de leads B2B e posicionamento de mercado em Angola. 

A arquitetura foi desenhada com um foco absoluto em **Performance** e **SEO Técnico**, garantindo tempos de carregamento instantâneos mesmo em redes 3G oscilantes, aplicando os princípios de resiliência que utilizo em sistemas SaaS de larga escala.

## 🎯 Filosofia Técnica: "Resilient-First"

Como estudante de Economia e Engenheiro de Software, o meu foco é transformar investimento técnico num benefício económico tangível. Os sistemas que construo (como o ecossistema **PharmaScan**) são desenhados para sobreviver à instabilidade da infraestrutura local sem perder a precisão dos dados.

**Destaques da Plataforma:**
* **Desempenho Extremo (Zero-JS):** Compilado em HTML estático via Astro para máxima velocidade e acessibilidade.
* **SEO "Santo Graal":** Schema.org (JSON-LD) injetado nativamente, tags canónicas e geo-referenciação (AO-HUA) para dominância em pesquisas orgânicas locais.
* **Design "Expensive & Editorial":** Sistema tipográfico premium (Playfair Display + Space Mono) com texturas *noise* para uma experiência de utilizador sofisticada.
* **Content Authority Layer:** Blog integrado nativamente com Markdown para publicação rápida de análises técnicas e estudos de viabilidade.
* **Lead Generation:** Integração serverless direta (Formspree) para captura e notificação instantânea de pedidos de orçamento.

## 🚀 Como Executar Localmente

Se desejas explorar a "cozinha" desta plataforma, podes correr o ambiente de desenvolvimento localmente:

1. **Clonar o repositório:**
   ```bash
   git clone https://github.com/Bruno-Correia-gize/clickKwanza-portfolio.git
   ```

2. **Navegar para o diretório:**
   ```bash
   cd clickKwanza-portfolio
   ```
3. **Instalar dependências:**
   ```bash
   npm install
   ```
4. **Iniciar o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
5. **Aceder:** Abre `http://localhost:4321` no teu navegador.
